# GoStudent Booking Widget Clone

A pixel-perfect clone of GoStudent's dynamic checkout page built with React 19, TypeScript, and Tailwind CSS v4. Features robust form validation (React Hook Form + Zod), responsive RTL/Arabic support, and seamless WordPress integration via a custom shortcode plugin.
